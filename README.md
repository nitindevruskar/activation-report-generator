# Activation Report Generator

A Python-based automation project that generates activation reports using Excel inputs and Supabase PostgreSQL.

## Technologies Used

- Python
- Pandas
- SQLAlchemy
- Supabase PostgreSQL
- OpenPyXL
- GitHub

## Features

- Automated Excel reporting
- Country-wise comparison
- Year-over-Year analysis
- Summary dashboards
- Professional Excel formatting

## Installation

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python report_generator_supabase_final.py
```
